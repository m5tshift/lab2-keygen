# keygen-and-patcher

## Keygen
С помощью декомпиляции файла в Ghidra была получена логика генерации лицензионного ключа, что позволяет нам воспроизвести алгоритм и получить keygen для этого приложения.

Генератор ключей выполняет следующие шаги:
1. Принимает идентификатор оборудования (HWID) в качестве входных данных
2. Вычисляет MD5-хэш от HWID
3. Меняет порядок байтов хэша на обратный
4. Преобразует перевернутый хэш в шестнадцатеричное строковое представление

## Использование

1. Сначала запустите `hack_package`, чтобы получить ваш HWID
2. Скопируйте отображаемый HWID
3. Запустите генератор ключей:
   ```bash
   python3 keygen.py
   ```
4. Вставьте ваш HWID
5. Программа выведет ваш лицензионный ключ

Пример:
```
Enter HWID: EB060900FFFBEBBF
License key: 9528cc35057706390c5be28ea7e56b98
```

## Patcher
```bash
python3 patcher.py
```
* файл должен быть в той же директории, программа не принимает аргументы
